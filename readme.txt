Proivides a gutenberg details and summary block wich can be used as a simple accordion type.

== Description ==
The plugin will provide a collapsible HTML <details> block with a <summary> element that will be shown when the element is collapsed. Inside the details you can put all possible editor blocks you want.

The plugin is provided without styling and relies on (child) theme developers to implement CSS for further visual styling.
